dataset.csv - основной итог

dataset.sql - сбор всех таблиц воедино, вместе с расчетом кол-ва объектов в радиусе от объявления  
objects_view.sql - приведение к единому формату всех таблиц с объектами инфраструктуры  
union.sql - приведение к единому формату и преобразование некоторых полей из датасетов, которые команда собрала по детальному парсингу каждой карточки объявления  

svb_notebook.ipynb - парсинг списка объявлений, нахождение координат для ник через геокодер, вытягивание полигонов Москвы, сопоставление координат с полигонами, скрипт просчет изохрон в пешей доступности  
Mall_Geolocation.ipynb - пример работы с геокодером, для определения координат тц  
new.py,parse_cian_2.ipynb - скрипты парсинг и обработки страницы объявления
